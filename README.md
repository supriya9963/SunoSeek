SunoSeek is an AI-powered, voice-based job search assistant designed for low-literate and digitally excluded workers.  
Users can simply speak in their local language through WhatsApp voice messages or IVR calls and instantly receive verified jobs in audio form.

Key Features
- 100% Voice Interaction ‚Äì No typing, no reading.
- Supports Hindi, Telugu, Tamil, Bengali, Marathi & more.
- Works on WhatsApp, 2G networks & feature phones (IVR).
- AI-powered job matching + simplified audio job summaries.
- Verified employers via NGO/government partnerships.
- Privacy-first design (No data shared without consent).

AI Components
- Whisper ASR (Speech-to-Text)
- NLP Intent Classification (Rasa / Transformers)
- Job Recommendation Engine
- AI Job Summarizer
- Multi-language TTS

Tech Stack
Frontend/UI: WhatsApp API, IVR (Exotel/Twilio)  
Backend: Python FastAPI / Node.js  
Database: MongoDB, PostgreSQL, Redis  
AI Models: Whisper, LLM-based NLP, Summarizer  
Hosting: Render / AWS / Railway
